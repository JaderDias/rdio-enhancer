rdio enhancer
=================


Current Features
================

* Album Menu Additons
  * Add Album to Playlist
	* Adds all tracks of the Album to a new or existing Playlist  
* Playlist Additions
  * Fork Playlist
    * Adds all tracks of the Playlist to a new or existing Playlist
  * Sort Playlist
    * Sort by Artist, Album, or Track Name
